## 一、搭建：GitHub Pages + Fork 模板

本博客基于 GitHub Pages 搭建，全程免费、无广告。核心思路是：**Fork（复制）一个现成的博客仓库，修改成自己的**。

### 1. 准备工作
- 注册 [GitHub](https://github.com/) 账号。

### 2. Fork 模板仓库
- 打开模板地址：[qiubaiying.github.io](https://github.com/qiubaiying/qiubaiying.github.io)
- 点击右上角的 **Fork**，将项目复制到你的账号下。

### 3. 重命名仓库（关键）
GitHub Pages 要求仓库名必须严格匹配用户名才能生效：
1. 进入你 Fork 后的仓库。
2. 点击 `⚙ Settings` -> `General`。
3. 找到 **Repository name**，修改为：你的用户名.github.io
> 例如我的用户名是 `zhou155166`，则填写 `zhou155166.github.io`。
4. 点击 **Rename**。

### 4. 开启 GitHub Pages
1. 进入仓库的 `Settings` -> `Pages`。
2. **Source** 选择 `Deploy from a branch`。
3. **Branch** 选择 `master`（或 `main`），目录选 `/ (root)`。
4. 点击 **Save**。

### 5. 访问博客
等待 1-3 分钟构建完成后，在浏览器访问：https://你的用户名.github.io
至此，博客框架搭建完成。
